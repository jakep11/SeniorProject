Paths

GET /user

Gets User objects. Optional query param of email determines limits response to Users with specified email or email prefix.

200 OK

Successful response

Response Example (200 OK) 
[
  {
    "email": "string",
    "id": "number"
  }
]

GET /topic

Gets a list of all Topics

POST /topic

Creates a new topic. Requires admin.
